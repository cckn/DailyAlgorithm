# [가장 큰 수](https://programmers.co.kr/learn/courses/30/lessons/42746)

## 풀이 방법
- 제일 큰 요소가 0이면 모든 요소가 0이므로 '0' 반환 
  - 제일 처음에 하는 이유는 리소스 낭비를 막기위함 
- 각 요소를 문자열로 변경 
- 순서에 따라 어떤게 더 큰 값이 되는지 비교하여 sort

## 막혔던 포인트 
- 주어진 두 문자열을 앞 뒤로 더해보고 더 큰수를 찾으면 되는데 그걸 생각 못해서 이리저리 헤맴
